Introducción a la Programación

4. Ciclo de vida del software

Sean cuales sean las fases en las que realicemos el proceso de desarrollo de software, y casi independientemente de él, siempre se debe aplicar un modelo de ciclo de vida.

Ciclo de vida del software

Es una sucesión de estados o fases por las cuales pasa un software a lo largo de su "vida".

El proceso de desarrollo puede involucrar siempre las siguientes etapas mínimas:

Etapa Descripción
Especificación y Análisis de requisitos Definir qué debe hacer el sistema
Diseño Planificar cómo se va a construir el sistema
Codificación Implementar el diseño en un lenguaje de programación
Pruebas Verificar que el sistema funciona correctamente
Instalación y paso a Producción Poner el sistema en funcionamiento
Mantenimiento Corrección de errores y mejoras continuas
